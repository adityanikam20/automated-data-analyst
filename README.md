📊 Automated Data Analyst
An AI-powered web application that automatically cleans, analyzes, and visualizes datasets with minimal effort.
---
🚀 Overview
This project is designed to simplify Exploratory Data Analysis (EDA).
Users can upload a dataset and instantly get insights, visualizations, and downloadable reports without writing code.
---
✨ Key Features
- 📂 Upload CSV or Excel files
- 🧹 Automatic data cleaning (missing values & duplicates)
- 📊 Interactive visualizations using Plotly
- 📈 Correlation analysis with heatmaps
- 🤖 Auto-generated dataset summary
- 🎯 Data quality score
- 📥 Download:
  - Filtered dataset (CSV)
  - JSON report
  - PDF report
  - Python visualization code
---
🛠 Tech Stack
- Frontend & UI: Streamlit
- Data Processing: Pandas
- Visualization: Plotly
- Reporting: ReportLab
- AI Integration: OpenAI / LLM
---
